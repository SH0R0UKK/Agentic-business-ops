# Gap Analysis Agent

**Framework:** LangGraph  
**Model Support:** Gemini, Perplexity Sonar  

---

## Overview

The Gap Analysis Agent is a specialized LLM-powered agent that identifies internal and market gaps for seed/pre-seed startups. It operates as a **worker agent** in a supervisor-worker architecture, receiving curated research and producing structured, evidence-backed gap reports.

**Core Capabilities:**
- Analyzes startup profiles against market benchmarks
- Identifies internal capability gaps (team, process, technical, resource)
- Identifies market positioning gaps (competitive threats, feature parity)
- Enforces evidence-based reasoning (all gaps must cite research)
- Filters low-confidence gaps and caps output for actionability
