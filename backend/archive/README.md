# Archive Folder

This folder contains deprecated, experimental, or unused files that have been moved here instead of being deleted.

## Why Archive Instead of Delete?

- Preserve historical context and experiments
- Allow easy restoration if needed
- Maintain audit trail of project evolution

## Contents

### Old Test Files
- Legacy test scripts that have been replaced by the unified pytest suite
- One-off experiments and debugging scripts

### Deprecated Code
- Old implementations that were replaced during refactoring
- Experimental features that were not adopted

### MLflow/DVC Artifacts
- Files related to removed MLflow and DVC integrations
- Can be safely deleted after confirming no dependencies

## Policy

Files in this folder are:
- Not imported or used by active code
- Not covered by tests
- Not part of the deployment
- Safe to delete after 90 days (if no issues arise)
